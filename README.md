# Bokun module for Prime CMS

Module to interact with Bokun web service.

## Available modules

* Booking
* Cart
* Customer
* Product

## Setup config

You should setup our API settings in APPPATH/config/bokun.php:

     return array(
          'endpoint'  => 'http://api.bokun.is',
          'accesskey' => '',
          'secretkey' => ''
     );
